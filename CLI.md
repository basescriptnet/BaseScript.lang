## â–¶ï¸ Getting started

> Install via npm

```sh
npm i basescript.js -g
```

> At any directory use

```sh
bsc -f "<file_name>"
```

> To alternatively run the cli

```sh
node cli --file "<file_name>" [--run?]
```

> Specify the output file

```sh
bsc -f "<file_name>" -o "<output_file_name>"
```

> For help use

```sh
bsc -h
# or
bsc --help
```

> Save to file, and then run from CLI

```sh
bsc -f "<file_name>" -r
```
